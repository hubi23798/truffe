## 1. The Mark

### Concept
The truffe.ai mark is a stylised cross-section of a black Périgord truffle — concentric organic rings radiating from a gold core on a dark brown body. The form is immediately legible as "truffle" while functioning cleanly as a brand mark at all sizes from 16px favicon to billboard.

### Algorithm
The mark is generated by the `truffleRingPath` function (seeded PRNG + Catmull-Rom bezier, 128 points per ring). Rings are irregular closed paths, not circles — each one uses multiple sine harmonics at different frequencies and phases to produce organic, non-repeating geometry.

### Parameters

| Parameter | Value | Notes |
|---|---|---|
| `bodyShape` | `squircle` | Soft-cornered square; reads "app icon" and "organic object" simultaneously |
| `bodyRound` | `100` | Maximum squircle curvature |
| `markColor` | `#2C1A0E` | Dark Périgord brown — the truffle body |
| `ringCount` | `7` | Dense enough to read as tissue layers, not a diagram |
| `ringMaxR` | `70` | Rings push to the body edge — fills the form |
| `ringInnerR` | `10` | Tight inner ring; leaves room for the core |
| `ringIrregularity` | `0.28` | Organic wobble without falling apart at small sizes |
| `ringHarmonics` | `3` | Sub-frequency bumps — the knobby surface texture of real truffle skin |
| `ringStroke` | interpolated `5px → 2px` | Outer rings heavier (skin), inner rings finer (flesh) |
| `ringColor` | `#F5E9D3` | Warm cream — the cut interior of a black truffle |
| `ringSeed` | `9021` | Fixed seed; produces a good organic pattern |
| `showCore` | `true` | |
| `coreColor` | `#C9A86A` | Warm gold |
| `coreR` | `8` | Slightly enlarged for legibility at small sizes |

### Stroke interpolation (code change)
In `truffleRings()`, replace the fixed `strokeWidth={t.ringStroke}` with a per-ring interpolation:

```js
const strokeWidth = N === 1 ? t.ringStroke : 5 - (3 * i / (N - 1)); // 5px outer → 2px inner
```

### Small-size variant (≤ 48px)
At favicon and small icon sizes, reduce detail to prevent visual noise:

| Parameter | Full (≥ 192px) | Small (≤ 48px) |
|---|---|---|
| `ringCount` | 7 | 3 |
| `ringHarmonics` | 3 | 1 |
| `ringIrregularity` | 0.28 | 0.20 |
| `ringStroke` | interpolated 5→2 | fixed 3 |

## 2. Wordmark

**"truffe.ai"** — three parts, one rhythm:

| Part | Style |
|---|---|
| `truffe` | Inter 800, `#0E0F0E` |
| `.` | Inter 800, `#C9A86A` (gold accent) |
| `ai` | Inter 800, `#0E0F0E` |

- Letter-spacing: `-3/1000em` (`-0.003em`)
- Line-height: `0.9`
- The gold dot is the only colour break — it links the mark's core to the wordmark

### Tagline
**"your money's second brain"**

- Font: JetBrains Mono 400
- Size: 20px (scales with wordmark)
- Tracking: `-0.04em`
- Colour: `#0E0F0E` at 35% opacity
- Displayed below the wordmark with a `-40px` gap (tight, overlapping baseline rhythm)

## 3. Color Tokens

```css
:root {
  --paper:   #F5F1EA;  /* Page background — warm off-white */
  --ink:     #0E0F0E;  /* Primary text */
  --truffle: #2C1A0E;  /* Mark body — dark Périgord brown */
  --gold:    #C9A86A;  /* Brand accent — wordmark dot, mark core, financial positives */
  --cream:   #F5E9D3;  /* Mark ring strokes */
  --forest:  #1F4D3A;  /* Secondary accent — interactive elements, links, hover states */
  --rule:    rgba(14,15,14,0.10); /* Borders and dividers */
}
```

### Accent usage
- **Gold (`--gold`):** wordmark dot, mark core dot, positive financial deltas (e.g. net worth up), call-to-action highlights
- **Forest (`--forest`):** buttons, links, hover states, nav active indicator
- These two are complementary, not interchangeable — gold = identity/brand moments, forest = action/interactive moments

### Relationship to existing UI spec
The prior Visual Refresh spec proposed teal (`#2D7D6F`) as the accent. Replace with:
- Teal → `--forest` `#1F4D3A` for interactive/action elements
- New `--gold` `#C9A86A` for brand identity moments

## 5. App Icon Integration

All existing icons (`public/icon-*.png`, `favicon.ico`, `apple-touch-icon.png`) use the old "B" mark and must be regenerated.

### Icon tile spec
App icons use the dark tile from `lockup.jsx` (`AppIcon` component):

| Asset | Dimensions | Tile bg | Mark color | Ring color | Core color | Ring variant |
|---|---|---|---|---|---|---|
| `favicon.ico` | 32×32 | `#0E0F0E` | `#F5E9D3` | `#C9A86A` | `#F5E9D3` | Small (3 rings) |
| `icon-192.png` | 192×192 | `#0E0F0E` | `#F5E9D3` | `#C9A86A` | `#F5E9D3` | Full (7 rings) |
| `icon-512.png` | 512×512 | `#0E0F0E` | `#F5E9D3` | `#C9A86A` | `#F5E9D3` | Full (7 rings) |
| `apple-touch-icon.png` | 180×180 | `#0E0F0E` | `#F5E9D3` | `#C9A86A` | `#F5E9D3` | Full (7 rings) |
| `icon-maskable-192.png` | 192×192 | `#0E0F0E` | `#F5E9D3` | `#C9A86A` | `#F5E9D3` | Full, mark at 60% of tile |
| `icon-maskable-512.png` | 512×512 | `#0E0F0E` | `#F5E9D3` | `#C9A86A` | `#F5E9D3` | Full, mark at 60% of tile |

On the dark tile background, mark colors invert from the standard palette: body becomes cream (`#F5E9D3`), rings become gold (`#C9A86A`). This is intentional — the dark background makes the standard dark-brown body invisible. The in-app nav mark (on a light background) uses the standard non-inverted palette.

### In-app nav mark
The nav wordmark uses inline SVG (not a raster image):

- Mark: 28×28px inline SVG, `markColor=#2C1A0E`, `ringColor=#F5E9D3`, small variant (3 rings)
- Wordmark: "truffe**.**ai" in Inter 800, 20px, `--ink` with gold dot
- Layout: `LockupHorizontal` pattern — mark left, wordmark right, `gap: 10px`
- Hover: wordmark dot transitions from `--gold` to `--forest`
